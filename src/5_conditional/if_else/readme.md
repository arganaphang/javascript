# Readme If-Else

## Logical Gate

### AND (&&)

| p     | q     | p && q |
| ----- | ----- | ------ |
| true  | true  | true   |
| true  | false | false  |
| false | true  | false  |
| false | false | false  |

### OR (||)

| p     | q     | p \|\| q |
| ----- | ----- | -------- |
| true  | true  | true     |
| true  | false | true     |
| false | true  | true     |
| false | false | false    |

### Not (!)

| p     | !p    |
| ----- | ----- |
| true  | false |
| false | true  |
